# 问题 09：性能回归检测缺失

## 严重程度
🟠 中等

## 问题描述

性能测试是手动运行的，没有自动化性能回归检测，无法及时发现性能退化。

## 问题详情

### 当前状态

- 性能测试需要手动运行
- 没有保存历史性能数据
- 没有性能回归检测
- 无法及时发现性能退化

### 问题分析

1. **性能退化无法及时发现**
   - 代码修改可能导致性能退化
   - 手动测试无法覆盖所有场景
   - 可能在发布后才发现性能问题

2. **无法追踪性能趋势**
   - 没有保存历史性能数据
   - 无法分析性能变化趋势
   - 无法确定性能优化效果

3. **无法对比不同版本**
   - 无法对比 v0.1.0 和 v0.2.0 的性能
   - 无法确定升级是否带来性能提升
   - 用户无法做出 informed 决策

## 影响

- 可能发布性能退化的版本
- 无法追踪性能优化效果
- 用户无法评估升级价值

## 改进建议

### 方案 A：使用 github-action-benchmark

```yaml
# .github/workflows/benchmark.yml
name: Benchmark

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  benchmark:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install Rust stable
        uses: dtolnay/rust-toolchain@stable

      - name: Run benchmarks
        run: cargo bench --bench all 2>&1 | tee output.txt

      - name: Store benchmark results
        uses: benchmark-action/github-action-benchmark@v1
        with:
          tool: cargo
          output-file-path: output.txt
          # 性能退化超过 10% 则报警
          alert-threshold: '110%'
          # 性能提升超过 10% 则标记为改进
          fail-on-alert: true
          # 保存到 gh-pages 分支
          auto-push: true
          gh-pages-branch: gh-pages
          benchmark-data-dir-path: dev/bench
```

### 方案 B：添加性能回归检测脚本

```python
#!/usr/bin/env python3
# scripts/check_perf_regression.py

import sys
import json
from pathlib import Path

def load_benchmark_results(file_path):
    """加载基准测试结果"""
    results = {}
    with open(file_path, 'r') as f:
        for line in f:
            if 'time:' in line:
                # 解析输出格式: test_name time: [123.45 ns 123.67 ns 123.89 ns]
                parts = line.strip().split()
                test_name = parts[0]
                time_value = float(parts[2])
                results[test_name] = time_value
    return results

def compare_results(old_results, new_results, threshold=1.10):
    """比较性能结果，检测回归"""
    regressions = []
    improvements = []

    for test_name, old_time in old_results.items():
        if test_name in new_results:
            new_time = new_results[test_name]
            ratio = new_time / old_time

            if ratio > threshold:
                regressions.append((test_name, old_time, new_time, ratio))
            elif ratio < 1 / threshold:
                improvements.append((test_name, old_time, new_time, ratio))

    return regressions, improvements

def main():
    if len(sys.argv) != 3:
        print("Usage: python check_perf_regression.py <old_results> <new_results>")
        sys.exit(1)

    old_file = sys.argv[1]
    new_file = sys.argv[2]

    old_results = load_benchmark_results(old_file)
    new_results = load_benchmark_results(new_file)

    regressions, improvements = compare_results(old_results, new_results)

    if regressions:
        print("❌ Performance Regressions Detected:")
        for test_name, old_time, new_time, ratio in regressions:
            print(f"  {test_name}: {old_time:.2f} ns -> {new_time:.2f} ns (+{(ratio-1)*100:.1f}%)")
        sys.exit(1)

    if improvements:
        print("✅ Performance Improvements:")
        for test_name, old_time, new_time, ratio in improvements:
            print(f"  {test_name}: {old_time:.2f} ns -> {new_time:.2f} ns (-{(1-ratio)*100:.1f}%)")

    print("✅ No performance regressions detected.")
    sys.exit(0)

if __name__ == "__main__":
    main()
```

### 方案 C：添加性能基准测试到 CI

```yaml
# .github/workflows/ci.yml
name: CI

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Run tests
        run: cargo test --all

      - name: Run benchmarks
        run: cargo bench --bench all 2>&1 | tee bench_output.txt

      - name: Check for performance regressions
        if: github.event_name == 'pull_request'
        run: |
          # 获取 main 分支的基准结果
          git fetch origin main
          git stash
          git checkout origin/main
          cargo bench --bench all 2>&1 | tee bench_main.txt
          git checkout -
          git stash pop

          # 比较结果
          python scripts/check_perf_regression.py bench_main.txt bench_output.txt
```

        }
    }
