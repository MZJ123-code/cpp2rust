### 方案 B：添加使用示例

```rust
// examples/hello_world.rs
//! Hello World 示例
//!
//! 演示基本的 REQ/REP 模式

use zmq_core::{Context, SocketType};

fn main() {
    // 创建上下文
    let ctx = Context::new();

    // 创建 REP socket
    let rep = ctx.socket(SocketType::REP).unwrap();
    rep.bind("tcp://127.0.0.1:5555").unwrap();

    // 创建 REQ socket
    let req = ctx.socket(SocketType::REQ).unwrap();
    req.connect("tcp://127.0.0.1:5555").unwrap();

    // 发送请求
    req.send(b"Hello", 0).unwrap();

    // 接收请求
    let msg = rep.recv_bytes().unwrap();
    println!("Received: {}", String::from_utf8_lossy(&msg));

    // 发送回复
    rep.send(b"World", 0).unwrap();

    // 接收回复
    let msg = req.recv_bytes().unwrap();
    println!("Received: {}", String::from_utf8_lossy(&msg));
}
```

## 性能优化

### 1. 使用 VSM 优化

```rust
// ✅ 推荐：小消息使用 VSM（零堆分配）
let msg = ZmqMessage::from_slice(b"hello"); // ≤30 字节

// ❌ 避免：大消息频繁分配
let msg = ZmqMessage::from_slice(&vec![0u8; 1024]); // >30 字节
```

### 2. 批量处理

```rust
// ✅ 推荐：批量发送
for i in 0..1000 {
    socket.send(&format!("msg {}", i), 0).unwrap();
}

// ❌ 避免：逐条发送后等待
for i in 0..1000 {
    socket.send(&format!("msg {}", i), 0).unwrap();
    // 不要在这里等待回复
}
```

## 错误处理

### 1. 使用 ? 操作符

```rust
fn send_message(socket: &Socket, data: &[u8]) -> Result<(), ZmqError> {
    socket.send(data, 0)?;  // ✅ 推荐
    Ok(())
}
```

### 2. 区分可恢复和不可恢复错误

```rust
match socket.send(data, 0) {
    Ok(_) => {}
    Err(ZmqError::Transport(TransportError::QueueFull)) => {
        // 可恢复：重试
        thread::sleep(Duration::from_millis(10));
        socket.send(data, 0)?;
    }
    Err(e) => {
        // 不可恢复：返回错误
        return Err(e);
    }
}
```
