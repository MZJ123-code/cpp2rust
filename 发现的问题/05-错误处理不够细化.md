# 问题 05：错误处理不够细化

## 严重程度
🟡 需要关注

## 问题描述

`ZmqError` 枚举可能过于笼统，缺少协议层面的详细错误码，错误恢复机制不明确。

## 问题详情

### 当前错误处理

```rust
// zmq-core/src/error.rs
pub enum ZmqError {
    // 可能的错误类型有限
}
```

### 问题分析

1. **错误类型不够细化**
   - 协议错误、编解码错误、安全错误混在一起
   - 无法精确区分错误来源
   - 不利于错误恢复

2. **缺少详细错误码**
   - 没有 ZMTP 协议错误码
   - 没有 POSIX 风格的错误码
   - 不利于调试和日志

3. **错误恢复不明确**
   - 不同错误是否可恢复？
   - 重试策略是什么？
   - 连接是否应该关闭？

## 影响

- 调试困难，无法快速定位问题
- 错误恢复策略不清晰
- 与 libzmq 的错误行为不一致

## 改进建议

### 方案 A：细化错误类型

```rust
use thiserror::Error;

#[derive(Error, Debug)]
pub enum ZmqError {
    #[error("codec error: {0}")]
    Codec(#[from] CodecError),

    #[error("protocol error: {0}")]
    Protocol(#[from] ProtocolError),

    #[error("security error: {0}")]
    Security(#[from] SecurityError),

    #[error("transport error: {0}")]
    Transport(#[from] TransportError),

    #[error("state error: {0}")]
    State(#[from] StateError),

    #[error("io error: {0}")]
    Io(#[from] std::io::Error),
}

#[derive(Error, Debug)]
pub enum CodecError {
    #[error("invalid greeting signature")]
    InvalidGreetingSignature,

    #[error("unsupported ZMTP version: {0}")]
    UnsupportedVersion(u8),

    #[error("invalid frame size: {0}")]
    InvalidFrameSize(usize),

    #[error("invalid command name: {0}")]
    InvalidCommandName(u8),
}

#[derive(Error, Debug)]
pub enum ProtocolError {
    #[error("missing required command: {0}")]
    MissingCommand(String),

    #[error("invalid state transition")]
    InvalidStateTransition,

    #[error("unexpected message")]
    UnexpectedMessage,

    #[error("protocol violation: {0}")]
    Violation(String),
}

#[derive(Error, Debug)]
pub enum SecurityError {
    #[error("invalid mechanism: {0}")]
    InvalidMechanism(u8),

    #[error("authentication failed")]
    AuthenticationFailed,

    #[error("encryption failed")]
    EncryptionFailed,

    #[error("decryption failed")]
    DecryptionFailed,
}

#[derive(Error, Debug)]
pub enum TransportError {
    #[error("connection refused")]
    ConnectionRefused,

    #[error("connection timeout")]
    ConnectionTimeout,

    #[error("address in use")]
    AddressInUse,

    #[error("network unreachable")]
    NetworkUnreachable,
}

#[derive(Error, Debug)]
pub enum StateError {
    #[error("socket not connected")]
    NotConnected,

    #[error("socket already connected")]
    AlreadyConnected,

    #[error("operation not permitted in current state")]
    InvalidState,

    #[error("message too large")]
    MessageTooLarge,
}
```

### 方案 B：添加错误码

```rust
/// ZMTP 协议错误码
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum ErrorCode {
    // 成功
    Ok = 0x00,

    // 协议错误
    ProtocolVersionMismatch = 0x01,
    InvalidMechanism = 0x02,
    MissingCommand = 0x03,
    InvalidFrame = 0x04,

    // 安全错误
    AuthenticationFailed = 0x10,
    AuthorizationFailed = 0x11,
    EncryptionFailed = 0x12,

    // 状态错误
    InvalidState = 0x20,
    MessageTooLarge = 0x21,
    QueueFull = 0x22,

    // 传输错误
    ConnectionRefused = 0x30,
    ConnectionTimeout = 0x31,
    NetworkUnreachable = 0x32,
}

impl ErrorCode {
    pub fn is_recoverable(&self) -> bool {
        match self {
            Self::ConnectionTimeout => true,
            Self::QueueFull => true,
            _ => false,
        }
    }
}
```

### 方案 C：添加错误恢复策略

```rust
/// 错误恢复策略
pub enum RecoveryStrategy {
    /// 重试指定次数
    Retry(u32),

    /// 指数退避重试
    ExponentialBackoff {
        initial: Duration,
        max: Duration,
        multiplier: f64,
    },

    /// 关闭连接
    CloseConnection,

    /// 丢弃消息
    DropMessage,

    /// 向上层报告错误
    Propagate,
}

impl ZmqError {
    pub fn recovery_strategy(&self) -> RecoveryStrategy {
        match self {
            Self::Transport(TransportError::ConnectionTimeout) => {
                RecoveryStrategy::ExponentialBackoff {
                    initial: Duration::from_millis(100),
                    max: Duration::from_secs(30),
                    multiplier: 2.0,
                }
            }
            Self::State(StateError::QueueFull) => {
                RecoveryStrategy::Retry(3)
            }
            Self::Protocol(ProtocolError::Violation(_)) => {
                RecoveryStrategy::CloseConnection
            }
            _ => RecoveryStrategy::Propagate,
        }
    }
}
```
