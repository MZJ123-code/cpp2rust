# 问题 03：安全机制不完整

## 严重程度
🟠 中等

## 问题描述

CURVE 安全机制和 ZAP 认证协议未完全实现和测试，无法在不可信网络中安全使用。

## 问题详情

### 当前安全机制状态

| 机制 | 实现 | 测试 | 状态 |
|------|------|------|------|
| NULL | ✅ | ✅ | 完成 |
| PLAIN | ✅ | ✅ | 完成 |
| CURVE | ✅ (feature-gated) | ❌ | 未测试 |
| ZAP | ❌ | ❌ | 未实现 |

### 问题分析

1. **CURVE 机制未测试**
   - 代码中已实现 CURVE 加密
   - 但没有单元测试验证正确性
   - 无法确认与 libzmq 的互操作性

2. **ZAP 认证协议未实现**
   - ZAP 是 ZeroMQ 的集中式认证协议
   - 支持自定义认证后端
   - 对企业部署至关重要

3. **安全隐患**
   - NULL 机制无加密，不适用于不可信网络
   - PLAIN 机制明文传输密码，不安全
   - CURVE 是唯一安全的机制，但未验证

## 影响

- 无法在不可信网络中安全使用
- 与 libzmq 的互操作性受限
- 企业部署场景无法满足安全要求

### 方案 A：完成 CURVE 机制测试

```rust
#[cfg(test)]
mod curve_tests {
    use super::*;

    #[test]
    fn test_curve_key_exchange() {
        // 测试 CURVE 密钥交换
        let server_key = CurveKey::generate();
        let client_key = CurveKey::generate();

        let server_mechanism = CurveServer::new(&server_key);
        let client_mechanism = CurveClient::new(&client_key, &server_key.public());

        // 测试握手
        let client_hello = client_mechanism.process(&[]).unwrap();
        let server_hello = server_mechanism.process(&client_hello).unwrap();
        let client_welcome = client_mechanism.process(&server_hello).unwrap();

        // 验证密钥交换成功
        assert!(server_mechanism.is_ready());
        assert!(client_mechanism.is_ready());
    }

    #[test]
    fn test_curve_encryption() {
        // 测试消息加密/解密
        let key = CurveKey::generate();
        let mechanism = CurveClient::new(&key, &key.public());

        let plaintext = b"hello world";
        let ciphertext = mechanism.encrypt(plaintext).unwrap();
        let decrypted = mechanism.decrypt(&ciphertext).unwrap();

        assert_eq!(&decrypted, plaintext);
    }
}
```

```rust
#[test]
fn test_curve_interop_with_libzmq() {
    // 测试与 libzmq 的 CURVE 互操作性
    // 需要启动 libzmq 进程
    let libzmq_port = start_libzmq_with_curve();

    let ctx = ZContext::new();
    let socket = ctx.socket(SocketType::REQ).unwrap();
    socket.set_curve_serverkey(libzmq_server_key).unwrap();
    socket.set_curve_secretkey(client_secret_key).unwrap();
    socket.connect(&format!("tcp://127.0.0.1:{}", libzmq_port)).unwrap();

    // 发送消息
    socket.send(b"hello", 0).unwrap();
    let response = socket.recv_bytes().unwrap();

    assert_eq!(response, b"world");
}
```
